# Учебно-методическое пособие по информатике (ИТМО)

LaTeX-проект для создания учебно-методического пособия по информатике для Университета ИТМО.

## Быстрый старт

```bash
# Сборка PDF
latexmk -pdf main.tex

# Очистка
latexmk -C
```

## Требования

- TeX Live / MacTeX
- Python 3 (для minted)
- Biber (для библиографии)

## Структура

```
Book/
├── main.tex                    # Главный файл
├── preamble/                   # Настройки пакетов
├── src/                        # Главы пособия
├── images/                     # Изображения
├── bibliography.bib            # Библиография
├── GOST.md                     # Требования ГОСТ
└── CODESTYLE.md                # Стиль кода
```

## Статус соответствия ГОСТ

### ✅ Реализованные требования

| Требование | Статус | Реализация |
|------------|--------|------------|
| Формат А4 | ✅ | `extarticle` |
| Шрифт 14 пт | ✅ | `\documentclass[14pt]{extarticle}` |
| Times New Roman | ✅ | `tempora + newtxmath` |
| Междустрочный интервал 1.5 | ✅ | `\onehalfspacing` |
| Кириллическая кодировка T2A | ✅ | `\usepackage[T1,T2A]{fontenc}` |
| Русский язык | ✅ | `\usepackage[english,russian]{babel}` |
| Нумерация страниц сквозная | ✅ | `\pagestyle{plain}` |
| Подпись рисунков снизу | ✅ | `\captionsetup[figure]{position=bottom}` |
| Подпись таблиц сверху | ✅ | Настроено в `caption` |
| Библиография по ГОСТ 7.1-2003 | ✅ | `biblatex + gost-numeric` |
| Сортировка библиографии ntvy | ✅ | `sorting=ntvy` |
| Нумерованные списки а), б), в)... | ✅ | `preamble/enumitem.tex` |
| Сноски постранично | ✅ | `\usepackage[perpage]{footmisc}` |
| Заголовки без точки | ✅ | `titlesec` |
| Висячие строки запрещены | ✅ | `\clubpenalty=10000`, `\widowpenalty=10000` |
| Переносы в заголовках запрещены | ✅ | Настроено в `titlesec` |
| Окружение «где» для формул | ✅ | Окружение `where` |
| Единицы измерения по ГОСТ | ✅ | `siunitx` с `locale=DE` |
| **Абзацный отступ 1.25 см** | ✅ | `\setlength{\parindent}{1.25cm}` |

### ⚠️ Частично реализованные требования

| Требование | Статус | Примечание |
|------------|--------|------------|
| Поля страницы | ⚠️ | Установлено 2.5 см со всех сторон. Требуется: левое 30 мм, правое 10 мм |
| Маркированные списки | ⚠️ | Используется тире (--) вместо дефиса (-) для лучшей типографики |

### ❌ Не реализованные требования

| Требование | Статус | Что требуется |
|------------|--------|---------------|
| Список сокращений | ❌ | Требуется добавить секцию |
| Приложения | ❌ | Требуется добавить окружение |

## Сборка

```bash
# Основная сборка
latexmk -pdf main.tex

# Принудительная пересборка
latexmk -pdf -g main.tex

# Очистка промежуточных файлов
latexmk -c

# Полная очистка
latexmk -C
```

Выходной файл: `build/main.pdf`
