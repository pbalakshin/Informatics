# Требования ГОСТ к оформлению учебно-методических материалов

## Обзор

Данный документ содержит требования к оформлению учебно-методических материалов для Университета ИТМО на основе действующих государственных стандартов (ГОСТ).

---

## 6. Оформление формул

### 6.1 Общие требования

| Требование                                                     | ГОСТ           | Пункт |
| -------------------------------------------------------------- | -------------- | ----- |
| Формулы располагаются отдельной строкой, по центру             | ГОСТ 7.32-2017 | 6.5.1 |
| Нумерация: арабскими цифрами в круглых скобках справа          | ГОСТ 7.32-2017 | 6.5.2 |
| Нумерация: сквозная или в пределах раздела                     | ГОСТ 7.32-2017 | 6.5.2 |
| Обозначение: (X) или (X.Y)                                     | ГОСТ 7.32-2017 | 6.5.3 |
| Расшифровка символов начинается со слова «где» (без двоеточия) | ГОСТ 7.32-2017 | 6.5.4 |
| Расшифровка располагается под формулой                         | ГОСТ 7.32-2017 | 6.5.4 |
| Ссылка в тексте: формула (X)                                   | ГОСТ 7.32-2017 | 6.5.5 |

**LaTeX:**
```latex
\[
    I = - \sum^{N}_{i=1}p_{i}\times \log_{2}{p_{i}},
\]
\begin{equation}\label{eq:shannon}
    E = mc^2
\end{equation}
\begin{where}
    \item [$I$] --- количество информации, в битах;
    \item [$N$] --- число состояний системы;
    \item [$p_{i}$] --- вероятность $i$-го состояния.
\end{where}
```

> **Важно!** Перед словом «где» в формулах требуется поставить запятую.

| Ссылки с указанием страницы: [X, с. Y]      | ГОСТ 7.0.5-2008 | —     |

### 7.2 Примеры библиографических записей

**Книга:**
```bibtex
@book{Cormen2013,
    author    = "Томас Х. Кормен and Чарльз Э. Лейзерсон and Рональд Л. Ривест",
    title     = "Алгоритмы: построение и анализ",
    edition   = "3",
    address   = "Москва",
    publisher = "ООО ''И. Д. Вильямс''",
    year      = "2013",
    pagetotal = "1328",
    language  = "russian",
}
```

**Электронный ресурс:**
```bibtex
@online{Top500,
    title    = "Top 500 supercomputers list",
    url      = "https://www.top500.org/",
    addendum = "(Дата обращения: 12.02.2023)",
    language = "english",
}
```

**LaTeX:**
```latex
\usepackage[
    backend=biber,
    style=gost-numeric,  % ГОСТ 7.1-2003
]{biblatex}
\addbibresource{bibliography.bib}

% В тексте:
Как показано в работе~\cite[с.~123]{Cormen2013}...

% Список литературы:
\printbibliography[title={Список использованных источников}]
```
